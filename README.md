# WebMarkets – Client Design Mockups

This repository hosts WebMarkets' client-facing design mockups via GitHub Pages. Each mockup is organized by client and can be shared directly via its hosted URL.

## Overview

There is no root homepage for this repository. Individual mockup pages are accessed directly by their URL path.

## Structure

```
/
├── client-name/
│   ├── project-name/
│   │   └── filename.html
│   └── ...
└── ...
```

## Accessing Mockups

All mockups are hosted at:

```
https://webmarkets.github.io/<client>/<filename>/
```

Share the direct URL with clients to give them access to their specific mockup.

## Contributing

1. Create a folder using the client and project name (use lowercase, hyphens for spaces).
2. Add your mockup files inside.
3. Push to `main` — GitHub Pages will deploy automatically.

## Notes

- This repo is intended for internal use and client sharing only.
- Avoid committing sensitive assets or credentials.
- If a client needs a mockup taken down, simply remove the folder and push.
